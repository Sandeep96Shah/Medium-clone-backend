## Apis

* signUp
* signIn
* Create a blog
* Save a blog
* View all the posted blogs

* View all the blogs

## Tech Stack

* nodejs(express)
* mongoose
* redis-server
* aws s3 bucket
* passport

## Run the project

* Clone the project
* run npm install
* npm start
* Visit  [http://localhost:8000](http://localhost:8000)
